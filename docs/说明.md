## 连锁
### 连锁处理逻辑
       while 玩家发动效果 {
            效果入栈
            if 效果是可以被连锁的 and 存在可发动和连锁的效果{
                通知玩家发动连锁
                if 没有玩家发动连锁 {
                    跳出循环
                }
            }
        }
        计算栈中的效果
        计算方法：
            1. 取出效果A
            2. 执行效果A
